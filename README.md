# Mars UAV Sizing

Feasibility study for a battery-powered QuadPlane UAV operating from a crewed Mars habitat.

## Project Structure

- `src/mars_uav_sizing/` - Python sizing tools
- `docs/` - Documentation and design specifications
- `sections_en/` / `sections_it/` - Manuscript sections (English/Italian)
- `figures/` - Generated figures

## Installation

```bash
uv sync
```

## Usage

```bash
uv run python -m mars_uav_sizing.run_sizing
```
