================================================================================
DATA FIELDS TO COLLECT
================================================================================

For each camera, collect ALL of the following fields.
Use null for any field where data is not available from the primary source.

| Field                | Description                              | Unit    |
|----------------------|------------------------------------------|---------|
| model                | Full product model name                  | string  |
| manufacturer         | Company name                             | string  |
| type                 | "rgb" or "multispectral" or "thermal"    | string  |
| mass_body_g          | Camera body mass only                    | grams   |
| mass_with_lens_g     | Mass including standard lens             | grams   |
| mass_total_system_g  | Mass with gimbal if commonly paired      | grams   |
| sensor_type          | e.g., "CMOS", "CCD"                      | string  |
| sensor_size          | e.g., "Full Frame", "1-inch", "4/3"      | string  |
| sensor_dimensions_mm | e.g., "35.9 x 24.0"                      | string  |
| resolution_mp        | Megapixels                               | number  |
| image_dimensions     | e.g., "9504 x 6336"                      | string  |
| lens_focal_length_mm | Focal length or range                    | string  |
| lens_fov_deg         | Field of view in degrees                 | number  |
| aperture_range       | e.g., "f/2.8 - f/11"                     | string  |
| shutter_speed_range  | e.g., "1/8000 - 30s"                     | string  |
| iso_range            | e.g., "100 - 51200"                      | string  |
| power_consumption_w  | Watts (if available)                     | number  |
| operating_temp_min_c | Minimum operating temperature            | number  |
| operating_temp_max_c | Maximum operating temperature            | number  |
| storage_temp_min_c   | Minimum storage temperature              | number  |
| storage_temp_max_c   | Maximum storage temperature              | number  |
| dimensions_mm        | "L x W x H"                              | string  |
| interface            | e.g., "USB-C", "Ethernet", "proprietary" | string  |
| video_capability     | e.g., "4K 60fps", "none"                 | string  |
| gsd_at_100m          | Ground sample distance at 100m AGL       | string  |
| source_url           | Primary manufacturer URL                 | string  |
| source_key           | Zotero citation key                      | string  |
| access_date          | Date accessed (YYYY-MM-DD)               | string  |
| notes                | Any relevant notes                       | string  |
