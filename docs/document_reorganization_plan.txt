================================================================================
MARS UAV FEASIBILITY STUDY - DOCUMENT REORGANIZATION PLAN
================================================================================
Date: 2025-12-26 (v3 - Final streamlined structure)
Status: APPROVED FOR IMPLEMENTATION

This plan reorganizes the manuscript for a feasibility study with:
1. Streamlined, flat structure (no unnecessary hierarchy)
2. Methodology presented upfront (after introduction)
3. Mission-driven design: payload first, then architecture context
4. All constraint inputs prepared before analysis
5. Aerodynamic analysis integrated with airfoil data
6. Appropriate scope for feasibility study (e.g., avionics recognized but not selected)

================================================================================
DESIGN PHILOSOPHY
================================================================================

The logical flow:
  1. Introduction (context and objectives)
  2. Methodology (how we approach the problem - sets expectations)
  3. Mission analysis (environment, profile, user needs)
  4. Reference data and trade-off analysis:
     - Payload first (mission-driven: UAV is built around it)
     - Architecture comparison (context for understanding designs)
     - Specific reference platforms and subsystem data
     - Analysis and synthesis (weight estimate, derived requirements)
  5. Constraint analysis (comparative, for three configurations)
  6. Design decisions (based on analysis results)
  7. Component selection and verification
  8. Infrastructure requirements
  9. Conclusions

CONTENT FOR THIS PROMPT:
Task: Research and document propulsion efficiency parameters for Mars UAV

Parameters to research:
1. Figure of Merit (FM) for small rotors at low-Re:
   - Search: Johnson Mars Science Helicopter [@johnsonMarsScienceHelicopter2020]
   - Search: Leishman for general rotor FM theory
   - Expected range: 0.6-0.7
   - Location in source required

2. Propeller efficiency (η_prop) at low Reynolds numbers:
   - Search: Bertani 2023 Mars propeller study
   - Search: McCormick "Aerodynamics, Aeronautics, and Flight Mechanics"
   - Expected range: 0.50-0.65
   - Note: efficiency degrades at Re < 100,000

3. Motor efficiency (η_motor):
   - Search: T-Motor MN505-S datasheet
   - Search: Commercial VTOL motor specifications
   - Expected range: 0.82-0.88

4. ESC efficiency (η_ESC):
   - Search: ESC manufacturer specifications
